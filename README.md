# 🔍 InvestigatorAI

**InvestigatorAI** is an intelligent assistant designed to help analysts, detectives, and security professionals process investigative data. It combines natural language understanding with structured reasoning to uncover hidden patterns, summarize findings, and suggest next steps.

## ✨ Features

- **Evidence analysis** – extract entities, timelines, and relationships from text logs, interviews, or reports.
- **Clue linking** – automatically connect related pieces of evidence across multiple sources.
- **Hypothesis generation** – propose possible scenarios based on known facts.
- **Case summarization** – generate concise, fact‑focused digests of long investigation files.
- **Chat‑based query** – ask natural‑language questions about your case data.

## 🧠 How it works

InvestigatorAI uses a combination of:
- Large Language Models (GPT‑4 / Llama / Claude) for reasoning and summarization
- Embedding‑based retrieval to find relevant evidence snippets
- Graph reasoning to visualize relationships between persons, places, and events
